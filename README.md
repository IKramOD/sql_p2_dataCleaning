# Nashville Housing Data Cleaning Project using SQL
Project Overview

This project focuses on cleaning and preparing the Nashville Housing dataset using MySQL. The goal is to transform raw housing data into a clean and structured dataset that can be used for further analysis and visualization.

The project demonstrates essential data-cleaning techniques commonly used by data analysts and data engineers.

Dataset

Dataset: Nashville Housing Data

The dataset contains information about property sales in Nashville, including:

Parcel IDs
Property and owner addresses
Sale dates and prices
Property characteristics (bedrooms, bathrooms, acreage, etc.)
Tax and property valuation information
Objectives

The objectives of this project were to:

Import data from a CSV file into MySQL.
Standardize inconsistent data formats.
Split and transform address information.
Standardize categorical values.
Remove duplicate records.
Eliminate unnecessary columns.
Prepare the dataset for future analysis.
